Программа RobotManager1
Предназначена для дистанционного управления роботом с определенной архитектурой
 (точнее в приложении), на которого загружено специальное программное обеспечение. 

Цель создания - платформа разработки простого интерфейса лоя управления роботами.

Робот обязательно должен подключаться к компьютеру через последовательный порт
и быть подключенным через заданный последовательный порт в момент обмена данными.

ВНИМАНИЕ! Пррограмма разработана исключительно для ОС Windows!
 Адаптаций под другие ОС нет!

Робот может в ответ на определенные запросы с компьютора
 выполнять определенные действия и посылать определенные сигналы.
Действия и сигналы робота зависят от архитектуры робота и его программного обеспечения.

Используя программу, пользователь может принимать сигналы робота
 и, в свою очередь отправлять запросы.

Программа представляет из себя окно 800 на 600 (размер не меняется), в котором расположены: поле фаила подсказок,
 кнопки "Установить новый" и "Сброс" для установки фаилов подсказок,
 окно "Порт" для установки портов ввода-вывода, кнопка "Обновить порты",
 строка ввода, кнопка "Отправить", 
 кнопки "Сохранить данные" и "Очистить поле вывода",
 поле статуса, окно вывода, окно с подсказкой.

Для начала работы нужно установить последовательный порт (далее порт), через который будет идти обмен данными.
 Если в окошке стоит "Не задан", значит порт не задан, связи нет.
 Нажмите на окошко и выберете порт, через который хотите производить обмен данными.
 Список портов составлен в момент последнего обновления.
 В списке есть те порты, которые на момент обновления были доступны и не заняты другими программами.
 Чтобы обновить список портов, нажмите кнопку "Обновить порты". 
ВНИМАНИЕ! После обновления портов текущий порт автоматически сбрасывается и становится "Не задан".
ВНИМАНИЕ! Если связь через порт по техническим причинам не осуществима,
 то обновления портов автоматически производиться НЕ БУДЕТ!

Для того, чтобы отправить команду роботу, нужно в строке ввода ввести команду или последовательность команд,
 после чего для отправки нажать "Отправить" или ввести "Enter". 
Последовательность команд нужно вводить через запятую. После запятой обязательно должен быть пробел.
Программа ликвидирует лишние пробелы.
ВНИМАНИЕ!! В УКАЗАНИИ К ПОДСКАЗКАМ ОЗНАКОМЬТЕСЬ С ОГРАНИЧЕНИЕМ НА КОМАНДЫ! ПРОГРАММА НЕ ЛИКВИДИРУЕТ ЭТИ ОШИБКИ!
После того, как команда (последовательность команд) успешно отправлена, программа ожидает ответа от робота.
Время ожидания ответа от робота не превышает двух секунд. 
После того, как получение ответа от робота завершено, в окне вывода появляется новая строка, 
где записана команда, двойное двоеточие, и результат ее выполнения.
Строка ввода автоматически очистится.
Если в процессе обмена данными возникла ошибка (по причине наличия ошибки в программе робота,
 обрыва связи или по какой-нибудь другой причине), в окне статуса
 и в результате выполнения появляется сообщение об этом.
Поле вывода можно очистить кнопкой "Очистить поле вывода".
Содержимое поля вывода можно сохранить в файл формата "txt",
 нажав кнопку "Сохранить данные" и выбрав тот файл, в который хотите записать данные.
ВНИМАНИЕ! Если вы записываете данные в уже существующий файл, то его содержимое будет безвозвратно утрачено,
 и не будет никакого предупреждающего окна!!

Команды можно отправлять в "Режиме свободной связи", или установив файл с подсказками.
Чтобы установить файл с подсказками, надо нажать кнопку "Установить новый"
 и выбрать нужный вам файл с подсказками.
Файл с подсказками должен удовлетворять определенным требованиям, описанным в приложении.
После установки фаила с подсказками в окне подмсказок появится список команд. 
В каждой строке будет записана команда и через двойное двоеточие - комментарий к ней.
Если вы установили файл подсказок, вы не сможете отправлять команды, которых нет в списке,
ограниченном файлом подсказок. Программа не отошлет команду, которой нет в списке подсказок,
 не отошлет все последующие команды, находящиеся в поле ввода, не очистит строку ввода
 и в поле статуса сообщит об ошибке с указанием, какой именно команды в списке не существует.
Для того, чтобы вернуться к "Режиму свободной связи", нужно нажать кнопку "Сброс". 
Тогда окно подсказок автоматически очистится, а ограничения на отправку команд будут сняты.
ВНИМАНИЕ! Файл подсказок может работать некорректно, если он составлен с ошибкой, или не для того робота,
 с которым производится обмен данными.
