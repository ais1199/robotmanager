Указание к прошивке пробота для программы RobotManager1.
Для платформы Arduino.
Связь с компьютером осуществляется через последовательный порт.
Подробное описание функций следует смотреть тут 
http://arduino.ua/ru/prog
или тут
https://www.arduino.cc

Скорость связи, устанавливаемая программой, равна 9600.
Для грамотного чтения/отправки данных следует поставить ту же скорость в Serial.begin(9600);

Чтение осуществляется с помощью функции Serial.read().
Проверить, пришла ли новая информация, можно с помощью функции  Serial.available().
ВНИМАНИЕ! Программа посылает роботу сообщения, содержащие не более 32 символов. 
ВНИМАНИЕ! Сообщение заканчивается символом '|', который добавляется программой дополнительно к команде.

Команды, которые может принимать робот, не должны содержать пробелов и двойных двоеточий.
Коменды не содержат параметров.
